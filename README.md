# Experimental DataBase
--------
### Проектная работа по курсу "Базы данных"

В этом репозитории находятся описание и скрипты для базы данных, моделирующей школьную физическую лабораторию. Целью базы данных является поддержание информации о количестве и составе установок, а также о проводимых занятиях и полученных оценках. Подробную информацию смотрите в файле `ExpDB_Description.pdf`.
